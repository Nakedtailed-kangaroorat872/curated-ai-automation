## 🧩 Terms You May See

You may see these terms in the repository:
- **AI automation**: uses AI to help with a task
- **No-code**: you build without writing code
- **Low-code**: you may use a few simple settings, but not full coding
- **Workflow**: a series of steps that run in order
- **Trigger**: the event that starts a workflow
- **Action**: the task that runs after the trigger
- **RPA**: software that copies human actions in apps
- **Open-source**: code that people can inspect and change
